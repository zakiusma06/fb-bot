NEW CORE WORKFLOW

Current logic saves one row per ad.

ONE ROW = ONE PRODUCT CLUSTER

Steps:

1. Bot finds an initial ad from Ads Library based on the user's search.
2. Open the ad landing page.
3. Extract probable product keywords from the landing page:
   - page title
   - product title
   - H1/H2 headings
   - repeated phrases
4. Generate a cleaner search keyword from that landing page.
5. Run a SECOND Ads Library search using those extracted keywords.
6. Collect ads that appear to sell the same product.
7. Build a product cluster.
