13. COST PER RESULT FIELDS
========================

Maintain these fields on:

- WINNER
- LOSER

Fields:
- 1 DAY COST PER RZLT
- 2 DAY COST PER RZLT
- TOTAL COST PER RZLT

Definitions:
1 DAY COST PER RZLT
- performance for first 24h from effective start time

2 DAY COST PER RZLT
- performance for second 24h window, from 24h to 48h

TOTAL COST PER RZLT
- total performance from launch until current/stop time

Formula:
- if results > 0 => spend / results
- if results = 0 => mark as NO RESULTS or equivalent safe value

Use the selected conversion event as the definition of “result”.

Also track:
- campaign age in hours
- first 24h checkpoint reached or not
- 48h checkpoint reached or not
